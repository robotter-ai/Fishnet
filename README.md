# Robotter.ai Frontend

## Setup

Clone the repository

Create .env file and paste the following:

```shell
VITE_ROBOTTER_API_URL = http://127.0.0.1:8000
VITE_TRANSACTIONS_API_URL = http://127.0.0.1:3000
```
## Install dependencies
```shell
yarn install
```
## Start Project
```shell
yarn dev
```
